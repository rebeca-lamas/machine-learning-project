Model Comparision

Both models, linear and logistic, are just approximations to the true unknown population of data points. While the multiple linear model is far easier to interpret and use, the logistic model captures the nonlinear nature of the population. There is no clear answer when it comes to selecting the correct model as the dataset used in the project may be lacking in key features (variables), specifically financial variables. These lack of variables may have given the multiple linear a better score. Overall, based off of the data collected, the logitstical regression is far better, statistically, than the mulitple linear model.
