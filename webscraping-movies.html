Webscraping Movies

The following websites were webscraped to create complete lists of movies released in the given year in the United States.

  • List of 2016 Movies
  • List of 2017 Movies
  • List of 2018 Movies
Beautiful Soup and pandas were used to parse the html and extract the title_ids needed to make our API calls. Since OMDb API limits the API calls allowed for their free tier to 1000 calls per day we batched the titles for each year into lists no longer than 1000 records. These were then exported to our dta folder. An example of the file name was "titles_2016_batch_1". Each year had four batches of titles.
